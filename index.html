Recuperação de face ID

três principais componentes do Face ID são
- Flood illuminator;
- Dot projector;
- Câmera frontal.
O Flood é o que estraga mais fácil, para concertar é necessário retira-lo do aparelho e colocar em uma máquina específica para copiar o código, após isso o código é copiado para um novo sensor Flood que em seguida é soldado no aparelho. Quando a manutenção é no sensor de iluminação, localizado no “flex” frontal, é feita a substituição da peça por meio de solda.
